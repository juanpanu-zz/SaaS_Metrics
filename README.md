# SaaS_Metrics
Web App using pandas plotly and Dash


![](assets/Capture.png)

### Prerequisites
```
$ pip install dash
$ pip install dash_core_components
$ pip install pandas
$ pip install numpy
$ pip install pandas
```
### Run
- Just run following command.
```
$ py saas_app.py
```
- Go to http://localhost:8050/
- Thats it.
